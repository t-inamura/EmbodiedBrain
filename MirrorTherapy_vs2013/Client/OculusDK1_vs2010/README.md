Name  
====  
OculusDK1_vs2010

## Overview  
 OculusRiftDK1で計測される首のオイラー角をサーバーへ送信します．
 KinectV2と同時に使用する場合は，こちらのサービスプロバイダを先に起動してくださ
 い（サーバ側のコントローラの実装の都合上，KinectV2が先に起動しているとOculus
 の結果を使用できません(要改善)）．

## Requirement
 ソリューション構成はDebugにしてください．

 新しい構造体を使用している以外は，
「C:\SIGVerse\GitHub\Client\Plugin\OculusRiftSensor\ViewerService」の
 OculusRiftSensor.sln の設定とほとんど同じだと思います．
